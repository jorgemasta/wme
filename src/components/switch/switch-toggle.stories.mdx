# Switch

A styled version of <a href="https://mui.com/material-ui/react-switch/" target="_blank">MUI's Switch</a>, so you can use any prop provided there.

To use with a label, pass in the `Switch` component to the `InputLabel` component as a prop, like:

```
<InputLabel
  control={<Switch />}
  label="I would like to receive updates"
/>
```

You can see more info in the <LinkTo kind="forms-inputlabel--base&args=control:switch">InputLabel documentation</LinkTo>

## Example

```
import React from 'react';
import { Switch } from '@moderntribe/wme';

export const MySwitch = (props) => (
  <Switch {...props} />
);

export default Switch;
```
