# Device Selection
Component displaying different devices that are clickable (used for changing screen size in previews). 

Takes an array of objects. Each object should be a device with a `name`, `icon`, and `onClick`. Here is an example:

```
export const devices = [
  {
    name: 'desktop',
    icon: DesktopWindows,
    onClick: () => console.log('desktop'),
    active: true,
  },
  {
    name: 'tablet',
    icon: TabletMac,
    onClick: () => console.log('tablet'),
  },
  {
    name: 'phone',
    icon: PhoneIphone,
    onClick: () => console.log('phone'),
  },
];
```
