# Buttons
Buttons allow users to take actions with a singular click.
They should be clear indications that they are triggering an action or advancing in a flow.
Buttons should always be easy to find among other UI.

## Usage
- Use 8 px spacing between buttons.
- Button width is set by the content it contains.
- Use only one primary button per screen. Any remaining actionable items should be presented with a lower emphasis call to actions.
