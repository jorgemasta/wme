# Field Label

A styled version of <a href="https://mui.com/material-ui/api/input-label/" target="_blank">MUI's InputLabel</a>, so you can use any prop provided there.

## Example

```
import React, { useState } from 'react';
import { FormFieldLabel } from '@moderntribe/wme';

export const MyFormFieldLabel = (props) => (
  <FormFieldLabel>Field Name<FormFieldLabel>
);

export default MyFormFieldLabel;
```
