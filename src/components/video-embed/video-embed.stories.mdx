# VideoEmbed
Basic video component that can be used in other components.
Height and width are set

<ArgsTable story="." />

## Example
```
import React, { useState } from 'react';
import { VideoEmbed } from '@moderntribe/wme';

const MyVideoEmbed = (props) => (
  <>
    <VideoEmbed src="https://www.youtube.com/embed/YwjYtoE5UMQ" height="500" width="700" />
  </>
);

export default MyVideoEmbed;
```
