# Setup Card Task CTA

Task Row CTAs are used in Setup cards for actions that will add tasks to a card.

Extends <a href="https://mui.com/material-ui/api/button/#main-content" target="_blank">MUI's Button</a>, but available props are limited for this component.

## Example

import { SetupCardTaskCta } from '@moderntribe/wme';
import DownhillSkiingIcon from '@mui/icons-material/DownhillSkiing';

const MySetupCardTaskCta = () => {
  const handleClick = () => {
    console.log('Click handled');
  }

  return (
    <SetupCardTaskCta 
      startIcon={
        <DownhillSkiingIcon />
      }
      onClick={handleClick}
    >Click Now</SetupCardTaskCta>
  );
};
