# Wizard Section Title
Each Wizard page requires a header with optional body copy to inform the user of the actionable items needed.

The Wizard Section Title Component extends MUI's box, so you can pass `sx` and other props if needed and they'll
be applied to the Section Title container.

## Props
<ArgsTable story="." />

## Bookend Section Headers
Bookend is a prop (boolean) that centers the entire element. It's used at the beginning and end of a Wizard. The user
is greated with a bookend header that features an optional icon, title, an body copy.

If bookend is set to true, it will override the `copyAlign` attribute.
