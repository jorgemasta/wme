# Modal
The Wizard/Modal is a Dialog box that can be opened/closed. It is the MUI `Dialog` component
wrapping the `DialogContent` component. Children are passed to `DialogContent`.

This Modal extends MUI's Modal, so you can access any props available there. <a href="https://mui.com/material-ui/api/dialog" target="_blank">You can view the reference here</a>.
The Modal will always expand to fullscreen. If you don't want the fullscreen Wizard option, we recommend using MUI's `Dialog` or `Modal` component instead.
