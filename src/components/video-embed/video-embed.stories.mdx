# VideoEmbed
Basic video component that can be used in other components.

Height and width are set using CSS instead of attributes on the iFrame.
These props should be passed as string values either as a percent or pixel
(i.e. '85%' or '250px').

## Example
```
import React, { useState } from 'react';
import { VideoEmbed } from '@moderntribe/wme';

  <>
    <VideoEmbed src="https://www.youtube.com/embed/YwjYtoE5UMQ" height="315px" width="560px" />
```
