# Text Field
An input field can include a label, text field users can type into, and helper text.

## Usage
- Always use a placholder in the input field
- Always provide a title for your input fields

## Props
<ArgsTable story="." />

## Example
```
import React from 'react';
import { TextField } from '@wme/components';

export const MyTextField = (props) => (
  <TextField
    placeholder="Placeholder text"
    helperText="Some helper text"
    label="My text field"
    error={props.error}
    errorMessage={props.errorMessage}
    endAdornment={
      <InputAdornment position="end">
        <AccountCircle />
      </InputAdornment>
    }
  />
)

export default TextField;
```
