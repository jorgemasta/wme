# Form

A styled version of <a href="https://mui.com/material-ui/react-box" target="_blank">MUI's Box</a>, so you can use any prop provided there.

## Example

```
import React from 'react';
import { Form } from '@moderntribe/wme';

export const MyForm = (props) => (
  <Form>
    <FormField
      label="Field Name"
      field={<TextInput />}
    />
  </Form>
)

export default Form;
```
