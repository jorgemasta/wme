# Progress Bar
Loading Bars are used to update users on background processes that require
no action from them until the action is complete.

## Example
```
import React, { useState } from 'react';
import { ProgressBar } from '@moderntribe/wme';

export const MyProgressBar = (props) => (
  <>
    <h1>Progress Bar</h1>
    <ProgressBar
      color="primary"
      variant="determinate"
      value={props.progress}
      statusMessage="Importing your colors..."
    />
  </>
);

export default MyProgressBar;
```
